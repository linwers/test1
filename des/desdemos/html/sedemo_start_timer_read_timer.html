Description

This model shows how to use the Start Timer and Read Timer blocks to measure entity-based delays. The Start Timer block starts a specified timer (equivalent to time-stamping an entity) for each arriving entity. The Read Timer block reads this timer for each entity and computes the delay that the entity incurred between the two timer blocks.
